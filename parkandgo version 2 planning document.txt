Park and Go вЂ“ Version 2 Planning Document
=================================================================
OVERVIEW
=================================================================
Version 2 of Park and Go is a complete architectural overhaul transforming the app from a Flask-based vanilla web app into a production-grade, cross-platform system. The backend migrates to FastAPI for async performance, while the frontend rebuilds in React + TypeScript, enabling a single codebase that deploys as both a website and a mobile app (PWA + Android APK).

Primary Goals:
- Migrate backend from Flask to FastAPI for async request handling
- Rebuild frontend from vanilla JS to React + TypeScript
- Single codebase for web and mobile (PWA + APK)
- Enhanced recommendation engine with modular ML-ready architecture
- Production-grade observability, monitoring, and performance optimization
- Clean, responsive UI/UX with loading states for async operations

=================================================================
KEY UPDATES FROM V1
=================================================================

1. Loading Indicators:
   - Skeleton loaders for initial page loads
   - Progress bars for operations exceeding 500ms
   - Spinner overlays for background calculations
   - Toast notifications for async task completion

2. UI/UX Enhancements:
   - Mobile-first responsive design (adapts to all screen sizes)
   - Simplified navigation with collapsible sidebar on mobile
   - Touch-friendly tap targets (min 44x44px)
   - Smooth transitions between app states
   - Pull-to-refresh on mobile views

3. Cross-Platform Architecture:
   - PWA configuration for installable web app
   - Capacitor/Cordova integration for APK generation
   - Responsive breakpoints for web/tablet/mobile
   - Platform-specific optimizations (touch vs click events)

4. Deployment Strategy:
   - Internal APK testing among friend group first
   - Progressive rollout based on feedback
   - No public Play Store release until validated

RECOMENDATION ENGINE
=================================================================
The recommendation engine ranks locations using a weighted scoring system. Each factor contributes a percentage of the final recommendation score. The total possible score is 100 points.

Scoring Rubric:
1. Cost(40): Prioritizes affordable options for users
2. Distance to campus(25): Rewards locations closer to campus
3. User Parking Preferences(15): Matches user-selected preferences
4. Major(10): Relevance based on the user's academic major
5. Verified Spot(10): Increases trust for verified locations


Score_cost = (
Cost - 40
distance to east/west campus - 25
preferences - 15
major - 10
verified spot - 10
)

Design Rationale
- Cost is the most influential factor because affordability is a primary decision driver.

- Distance strongly impacts convenience for students traveling between campuses.

- Preferences allow personalization within the recommendation results.

- Major provides contextual relevance but is not a dominant signal.

- Verified spots improve reliability and user trust in recommendations.

  const map = new maplibregl.Map({
      style: `https://tiles.openfreemap.org/styles/bright`,
      center: [-74.0066, 40.7135],
      zoom: 15.5,
      pitch: 45,
      bearing: -17.6,
      container: 'map',
      canvasContextAttributes: {antialias: true}
  });

  // The 'building' layer in the streets vector source contains building-height
  // data from OpenStreetMap.
  map.on('load', () => {
      // Insert the layer beneath any symbol layer.
      const layers = map.getStyle().layers;

      let labelLayerId;
      for (let i = 0; i < layers.length; i++) {
          if (layers[i].type === 'symbol' && layers[i].layout['text-field']) {
              labelLayerId = layers[i].id;
              break;
          }
      }

      map.addSource('openfreemap', {
          url: `https://tiles.openfreemap.org/planet`,
          type: 'vector',
      });

      map.addLayer(
          {
              'id': '3d-buildings',
              'source': 'openfreemap',
              'source-layer': 'building',
              'type': 'fill-extrusion',
              'minzoom': 15,
              'filter': ['!=', ['get', 'hide_3d'], true],
              'paint': {
                  'fill-extrusion-color': [
                      'interpolate',
                      ['linear'],
                      ['get', 'render_height'], 0, 'lightgray', 200, 'royalblue', 400, 'lightblue'
                  ],
                  'fill-extrusion-height': [
                      'interpolate',
                      ['linear'],
                      ['zoom'],
                      15,
                      0,
                      16,
                      ['get', 'render_height']
                  ],
                  'fill-extrusion-base': ['case',
                      ['>=', ['get', 'zoom'], 16],
                      ['get', 'render_min_height'], 0
                  ]
              }
          },
          labelLayerId
      );
  });
</script>

- Mobile: Capacitor
